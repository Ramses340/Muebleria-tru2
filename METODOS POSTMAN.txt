| # | Acción                         | Método | URL                  | Token requerido |
| - | ------------------------------ | ------ | -------------------- | --------------- |
| 1 | Registrar usuario              | POST   | `/api/auth/register` | NO               |
| 2 | Iniciar sesión (obtener token) | POST   | `/api/auth/login`    | NO               |
| 3 | Listar productos               | GET    | `/api/productos`     | SI               |
| 4 | Crear producto                 | POST   | `/api/productos`     | SI               |
| 5 | Actualizar producto            | PUT    | `/api/productos/:id` | SI               |
| 6 | Eliminar producto              | DELETE | `/api/productos/:id` | SI               |
| 7 | Obtener producto individual    | GET    | `/api/productos/:id` | SI               |
